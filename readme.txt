Technologies required :
xampp(lamp)
Web Browser

Instructions to run/test the Feedback Management Portal :

0) Install xampp on the system
1) Copy the feedback_portal folder and Paste it in C:\xampp\htdocs (or wherever the htdocs folder is present)
2) Run the xampp control panel and start the MySql and Apache server
3) Open a web browser and type "localhost/phpmyadmin" in the URL and press enter
4) Create a new database/schema with the name : dkml
5) Import the database from the file "dkml.sql" in the folder feedback_portal 
6) Open a web browser and type the following URL : 
     localhost/feedback_portal/index.php
7) The required settings for the MySql connection are : 
	username : "root"
	password : ""  (double quotes for clarity -password is empty)
